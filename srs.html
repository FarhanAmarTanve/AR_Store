1. Introduction

1.1 Document Purpose

Augmented Reality is considered as a promising technology. This technology is offering an AR experience to the users with immersive solutions in various verticals. Likewise, Augmented Reality for the furniture industry is introducing new opportunities for furniture businesses. They will be able to sell their products in an effective way. AR is enabling the users to get more in-depth details on the object. This is helping the businesses in furniture to transform their business prospects

1.2 Product Scope

With the support AR, it easy to broadcast hundreds of furniture designs and thousands of colors. Moreover, AR in furniture has taken a whole new avatar of shopping experience. This persuasive technology is currently used for numerous reasons in different industries. And one industry which is using this technology to lift its potential is the Furniture Industry. The furniture industry is leveraging this advanced technology to design and develop complex designs. AR applications in furniture can offer benefits in the planning and design of home décor.

Today, Augmented Reality has a larger potential application in various verticals. AR solutions are easier to fit into our daily lives due to the utilization of smartphones. This is helping retailers can use this technology in multiple ways.

  • Businesses in the furniture industry can use AR to overcoming physical barriers.
  • The AR applications can help users to virtually try on furniture to their smartphone with personalized instructions.
  • AR can help to improve the retention rate of users with superimposing products with real-life environments.
  • Help to reduce pre-purchase indecision while cutting the rate of product return post-purchase.
  • AR also enables to easily personalize the product offering tailored promotions based on personal preferences.
  • Using analytics tools customer preferences can be gathered in different ways forecasting on the basis of the user’s history.
  • Furthermore, AR helps to create a unique customer journey in order to stay a step ahead of the competition.

1.3 Intended Audience and Document Overview

Augmented Reality is helping businesses to educate users about the products and engage them meaningfully. AR in furniture is a smart way to connect potential users in a unique way. The AR apps provides users with an advanced and immersive shopping experience. With AR, customers can browse the company’s furniture collection using their smartphones in their homes. It creates a more streamlined, enjoyable shopping user experience in a whole new way to transform your business.

1.4 Defintions, Acronyms and Abbreviations

Augmented Reality (AR) is a live or indirect view of a physical,real-world environment whose elements are "augmented" by computer-generated or extracted real-world sensory input such as sound, video, graphics, haptics or GPS data.

Framework is a real or conceptual structure intended to serve as a support or guide for the building of something that expands the structure into something useful.

2. Overall Description

2.1 Product Overview

Our system consists of an online furniture shopping which includes an extra feature called live view. This live view feature acts as a medium between our app and the web. This will allow the buyers/users to see the product virtually on their mobile/tablet devices using the camera function projecting an image of the product on the screen which they can adjust as they like. Each user can add products form their shopping cart and drag it to the display screen.

2.2 Product Functionality

The major function of our project is to create an online furniture store which allows user to login to the page to gain access to feature like live view. This feature allows users to use our app on the phone to not only customise the product but also see their creation in a 3- dimensional environment like their surroundings on their devices and allowing them to add even more product in that environment.

2.3 Design and Implementations Constraints

  • Required to complete a functional app and web-based product within 3 months.
  • Required to have a front end developer and back end developer and an Android Developer.
  • Must be able to work multiplatform.

2.4 Assumptions and Dependencies

  • Client or User has an active Internet Connection.
  • Client or User runs an operating system with a internet browser
  • Each user must have their own USER ID or Username and Password
  • Only one admin

4. Other Non-functional Requirements

4.1 Performance Requirements

System’s visual must run smoothly without any latency to keep the level of immersion high. This requirement is depended on many aspects of the user mobile device/tablet. System will be developed by using Unity3D and Vuforia; therefore, versions of Unity3D and Vuforia must support the mobile devices’/tablets’ OS. Unity3D version must be the latest version (2017.2+). For Vuforia to be worked on mobile devices/tablets; Android OS must be 4.1.x+, IOS must be 9+.

4.2 Safety and Security Requirements

  • Unauthorized access to the system and its data is not allowed.
  • Ensure the integrity of the system from accidental or malicious damage.
  • The access of permission for the system data may only be changed by the system's data admintrator.
  • All system data must be backed up every 24 hours and the back up copies stored in a secure location which in not in same building as the system.
  • All external communcations between the systems data server and clients must be encrypted.

4.3 Software Quality Attributes

4.3.1 Portability

Augmented Reality Based Continuous Onboarding Framework is designed for mobile devices/tablets using Unity3D and Vuforia. Unity3D supports many of the mobile devices/tablets, so system can be used mobile devices/tablets if their OS versions are supported.

4.3.2 Performance

he distance between the camera and the image shall not be more than 0.5 meters to scan. Scan operation shall not be last more than 10 seconds.

4.3.3 Usability

The usability should be as simple as possible since the screen of a mobile device is not big enough to place many information. New users are able to learn using the system in 5-15 minutes.
